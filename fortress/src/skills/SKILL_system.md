# SystemSkill — פקודות מערכת

## תיאור
פקודות מערכת: ביטול פעולה, אישור פעולה, עזרה.

## פקודות

| תבנית | פעולה | דוגמה |
|--------|-------|-------|
| `לא` / `עזוב` / `בטל` / `cancel` | ביטול | לא |
| `כן` / `yes` / `אישור` / `ok` | אישור | כן |
| `עזרה` / `help` / `פקודות` | עזרה | עזרה |

## הערות
- ביטול ואישור מזוהים ע"י CommandParser (לא ע"י הסקיל עצמו)
- עזרה מציגה את כל הפקודות הזמינות מכל הסקילים
