# TaskSkill — ניהול משימות

## תיאור
ניהול משימות משפחתיות: יצירה, רשימה, מחיקה, השלמה, עדכון.

## פקודות

| תבנית | פעולה | דוגמה |
|--------|-------|-------|
| `משימה חדשה: <כותרת>` | יצירת משימה | משימה חדשה: לקנות חלב |
| `new task: <title>` | יצירת משימה | new task: buy milk |
| `משימות` / `tasks` | רשימת משימות | משימות |
| `מחק משימה <מספר>` | מחיקת משימה | מחק משימה 2 |
| `מחק הכל` | מחיקת כל המשימות | מחק הכל |
| `סיים משימה <מספר>` | השלמת משימה | סיים משימה 1 |
| `עדכן משימה <מספר>` | עדכון משימה | עדכן משימה 3 |

## הרשאות
- קריאה: `tasks.read`
- כתיבה: `tasks.write`

## הערות
- מחיקה דורשת אישור (כן/לא)
- זיהוי כפילויות: מדויק, תת-מחרוזת, ונורמליזציה עברית
- מספרי משימות מתייחסים לרשימה האחרונה שהוצגה
