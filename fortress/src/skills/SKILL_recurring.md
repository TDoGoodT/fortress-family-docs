# RecurringSkill — תזכורות חוזרות

## תיאור
ניהול תזכורות חוזרות: יצירה, רשימה, ביטול.

## פקודות

| תבנית | פעולה | דוגמה |
|--------|-------|-------|
| `תזכורת חדשה: <כותרת>, <תדירות>` | יצירת תזכורת | תזכורת חדשה: ארנונה, חודשי |
| `recurring: <title>` | יצירת תזכורת | recurring: rent, monthly |
| `תזכורות` / `חוזרות` / `recurring` | רשימת תזכורות | תזכורות |
| `מחק תזכורת <מספר>` / `בטל תזכורת <מספר>` | ביטול תזכורת | מחק תזכורת 2 |

## תדירויות
- `יומי` → daily
- `שבועי` → weekly
- `חודשי` → monthly (ברירת מחדל)
- `שנתי` → yearly

## הרשאות
- קריאה: `tasks.read`
- כתיבה: `tasks.write`
