# ChatSkill — שיחה חופשית וברכות

## תיאור
ברכות דטרמיניסטיות (ללא LLM) ושיחה חופשית עם AI.

## פקודות

| תבנית | פעולה | דוגמה |
|--------|-------|-------|
| `שלום` / `היי` / `hello` / `בוקר טוב` / `ערב טוב` / `לילה טוב` | ברכה | שלום |
| כל הודעה שלא תואמת פקודה אחרת | שיחה חופשית | מה המצב? |

## הערות
- ברכות הן דטרמיניסטיות — ללא קריאת LLM
- שיחה חופשית עוברת דרך LLM (Bedrock/OpenRouter/Ollama)
- PII מוסר לפני שליחה ל-LLM ומשוחזר בתשובה
- זיכרונות נטענים כהקשר לשיחה
